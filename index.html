Welcome to ArnixOS!


The fast and lightweight OS

Packages | Documentation | Github

Features:-

  1. Fast
  2. Lightweight because no bloat. Only the needful stuff
  3. Good for developers as programming languages like python , nim , dart , nodejs etc and various libraries are in the standard repos
  4. PCs no longer have to go to the dumpster because with ArnixOS your pc can get a second life
  5. Best Windows Program compatibility with wine
  6. ArnixOS Supports Valve's Steam Play(Compatibility Layer for Windows Games) , PCSX2(PS2 Emulator) , XEMU(XBOX Emulator) , MGBA(GBA Emulator) etc..
  7. ArnixOS uses the Arch Linux repos which have more than 10000 software and with the AUR one can extend to more than 80000 +
  8. Fully under your control

Preinstalled Software:-

  • Firefox 102.7.0 ESR(Extended Supported Release). It will run better on older hardware with latest updates.
  • Linux Zen Kernel 6.1.11 giving the best performance.
  • Octopi making it easy to install software from the repository and the AUR
  • Application Suite(Text Editor , Image Viewer)etc are trusted as they are made by ArnixOS Devs

Minimum System Requirements:-

  • Hard Disk Space:- 6GB(Minimal) , 15GB(Recommended)
  • CPU:- Any processor having 1.6Ghz and x86_64, 2Ghz(Recommended for web browsing)
  • Memory:- 768MB(for minimal usage) , 2GB(For web browsing)
  • An internet connection(for installation) either wired or wireless
Please ensure that your machine meets the above requirements so that you can install
